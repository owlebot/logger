# template

Template repo for future node projects

## Setup

Node version: [Node v18+](https://nodejs.org/en/download/)

**Requirements:**  
Copy `secret.template.json` as `secret.json`.  

**Install dependencies:**  
`yarn`  

**Start application:**  
`yarn start`
